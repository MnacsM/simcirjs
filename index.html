SimcirJS

SimcirJS(a.k.a. Simcir) is a circuit simulator in HTML5 and JavaScript.
Let's try your circuit!
Kazuhiko Arase 様作成の SimcirJS をベースに，使用パーツの数を調整したものです．

Usage

  • Choose a device from the toolbox and move to right side.（ツールボックス（左側）からデバイスを選び，右側のワークスペースに移動させます．）
  • Connect them by drag operation.（デバイス同士をドラッグ操作で接続します．）
  • Click an input node to disconnect.（入力ノード（端子）をクリックすると，接続を解除できます．）
  • Move a device back to the toolbox if you don't use.（使わないデバイスはツールボックスへドラッグして戻すことで削除できます．）
  • Ctrl+Click(Mac:command+Click) to toggle view (Live circuit or JSON data).（Ctrl + クリック（Macは Command + クリック）で，回路の表示モード（回路図 ⇔ JSONデータ）を切り替えることができます．）
  • Double-Click a label to edit device name.（ラベルをダブルクリックすると，デバイス名を編集できます．）
  • v get data v ボタン: 現在配置されている回路の状態を下のテキストエリアに JSON データとして出力します．
  • ^ set data ^ ボタン: 下のテキストエリアにある JSON データを読み込んで回路図として画面上に復元します．
  • v get SVG v ボタン: 現在の回路図を SVG コードに変換して下のテキストエリアに出力し，同時に SVG ファイル（circuit.svg）としてダウンロードします．
